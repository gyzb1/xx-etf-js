# 低波红利ETF - 东方财富

基于东方财富数据的A股低波红利ETF组合系统，可部署到Vercel。

## 功能特性

- 智能选股：市值前30% + 高股息率前50只
- 等权重配置：每只股票2%权重
- 实时数据：基于东方财富公开API
- 现代化UI：Next.js + React + TypeScript
- 无需API密钥：使用东财公开接口，无访问限制

## 本地开发

1. 安装依赖：

```bash
npm install
```

2. 启动开发服务器：

```bash
npm run dev
```

3. 打开浏览器访问 `http://localhost:3000`

## Vercel 部署

1. 推送代码到 GitHub
2. 在 Vercel 导入仓库
3. 无需配置环境变量
4. 点击部署

## API 接口

- `GET /api/div-lowvol` - 获取低波红利组合
  - 参数：
    - `topN`: 选取股息率最高的前N只（默认50）
    - `percentile`: 市值百分位（默认0.3，即前30%）

## 选股策略

1. **市值筛选**：按总市值排序，选取前30%的大市值股票
2. **股息率筛选**：从大市值股票中，选取股息率最高的前50只
3. **等权重配置**：每只股票分配相同权重（2%）

## 技术栈

- Next.js 14 (App Router)
- React 18
- TypeScript
- 东方财富公开API
